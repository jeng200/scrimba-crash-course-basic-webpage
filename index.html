We use CSS to add style to our HTML documents.

There are many ways to add CSS to a document, but for the time being we will be using inline CSS.

Inline CSS

Inline CSS is written directly in our HTML using a style attribute.

We can style any element using inline styles, from the body to headings to paragraphs, as well as links and strong and em tags too!
